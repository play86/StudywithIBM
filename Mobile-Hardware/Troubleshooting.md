# Troublshooting network connectivity issues 👨‍💻🌐

Proceed to: `Rule out the cilent side` & `Environmental conditions for wirleless networks`

  ### Rule out Cilent side ( for wired & wireless)
  -  Check The Hardware
  -  Check the Network adapters

  ---  Routers assign IPs to network adapters
      wireless: confirms network adapter is enabled and set to automatically connect to the known network 
      Wired: Ensures betwork adapters are set to obtain IP automatically
      ⚠️ Always remember to check for outdated network drivers and update them.
  -  Check workplace Software for conflicts
  -  Check the Firewall
  ### Environmental conditions for wireless networks 
  -  Think about the positioning of the routers & Walls
  -  think about the signal interference
  -  Wifi congestion must be solved
